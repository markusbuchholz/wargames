# Wargames

---

## Build 

```bash

cd docker

sudo ./build.sh

```


## Run Mesa Exaples

```bash

cd docker

sudo ./run_app.sh
 
```

## Run Mesa App

```bash

cd docker

sudo ./run.sh

solara run app.py --host=0.0.0.0 --port 8765

```
